# garch
# 基于GARCH（异方差时间序列模型）的价格预测

## 课题任务的内容和要求（包括原始数据、技术要求、工作要求等）：
1.设计基于GARCH的虚拟机价格预测方法
2.开发可视化的Java程序界面，满足数据的加载和预测结果的展示

## 课题成果的要求（包括毕业设计论文、图表、实物样品等）：
1.算法结果要在大规模数据上进行测试，并绘制相关的图表。
2.JAVA结果展示程序要能够导入数据并进行预测结果展示。
